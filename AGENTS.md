# AGENTS.md

## Preview Rule

- After any page-related change, always open a local preview for the user.
- For this project, use `python3 -m http.server 8001` from the repository root if a local server is needed.
- After starting the local server, open the relevant page in the browser so the user can immediately review the change.
- If the relevant page is already open in preview, do not open it again; just tell the user to refresh.
